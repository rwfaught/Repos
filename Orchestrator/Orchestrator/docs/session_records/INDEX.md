# Session Records Index

## Purpose

This index is a lightweight discovery aid for repo-backed session records. It
helps locate named records without loading the full `docs/session_records/`
folder by default.

Session records remain `READ_WHEN_NAMED` evidence artifacts. This index is not
startup authority, not a CTO handoff, and not ratification of any listed
record.

## Index Rules

- Keep entries compact.
- Add or update entries only under an explicit boundary.
- Link to records by filename.
- Do not paste full closeout reports into this index.
- Do not use this index to route the next session.
- Do not use this index as a replacement for current repo docs,
  CTO/coordinator review, or accepted coordination docs.

| Date | Role | Boundary | Record | Status | Notes |
| --- | --- | --- | --- | --- | --- |
| YYYY-MM-DD | role | boundary-name | `filename.md` | evidence-only | one-line scope note |
